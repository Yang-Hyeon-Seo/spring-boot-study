# AOP(Aspect Oriented Programming, 관점지향 프로그래밍)
## OOP(객체지향 프로그래밍)
- 자바에서는 재사용 가능한 코드를 만들기 위해 클래스를 작성하고, 클래스를 사용해 필요한 만큼 객체를 만들어 사용할 수 있음
- 공통된 기능은 부모 클래스로 작성해 클래스가 상속하도록 지원함
## AOP(관점지향 프로그래밍)
- 자바 프로그래밍 언어는 이렇게 부모 클래스를 자식 클래스가 상속 받는 수직적인 방법 이외에 상속 없이 수평적으로 공통의 관심사를 구현하는 방법도 제공함 → 이것이 관점지향 프로그래밍
- 스프링 부트에서는 관점 지향 프로그래밍을 좀 더 쉽고 편리하게 구현할 수 있도록 스프링 AOP를 제공함
